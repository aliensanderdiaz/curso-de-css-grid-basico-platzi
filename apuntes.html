Apuntes - Curso de CSS Grid Básico

Sección 1 de 13 - ¿Qué es CSS Grid Layout?

Es una especificación de CSS, que nació por la necesidad de hacer layouts mucho mas dinámicas.
Antes solo se usaban tablas o flotantes y eso no permitia crear todas las estructuras que se querían ver en la web.
Css Grid implementa un sistema de grilla, que es una cuadricula que tiene filas y columnas.
Así podemos jugar con esos ejes para darle estructura a nuestra página.

Sección 2 de 13 - Conceptos para comenzar

CONTENEDOR: Es el elemento que se va a convertir en una grilla.
ITEMS: Son los elementos de la grilla.
LINEA: Son los elementos que limitan o dividen una grilla. Tener en cuenta que hay lineas horizontales y verticales.
CELDA: Es la unidad minima que podemos tener en una grilla. Esta limitada por 4 lineas. Normalmente ocupa una fila y una columna.
GRUPOS DE CELDA: Pueden ser: Track ó Area
TRACK: Ocupan una sola fila o una sola columna.
AREA: Ocupan mas de una fila o mas de una columna.

Sección 3 de 13 - Propiedades del contenedor.

  • Display grid
  • Grid-template
  • Gaps
  • Grid-auto

                                <div class="contenedor">
                                    <div class="item">1</div>
                                    <div class="item">2</div>
                                    <div class="item">3</div>
                                    <div class="item">4</div>
                                    <div class="item">5</div>
                                    <div class="item">6</div>
                                </div>
                            

                                .contenedor {
                                    border: 5px solid #e1bee7;
                                    background-color: #fff1ff;
                                }
                                
                                .item {
                                    border: 5px solid #00bcd4;
                                    font-size: 4rem;
                                }

Vamos a la práctica, aquí vamos a ver 6 cajas dentro de una caja.
Aquí no hay ninguna magia.
Pero vamos a convertir la caja grande en un contenedor. 


                                .contenedor {
                                    ...
                                    display: grid;
                                }

Con solo esta propiedad grid ya hemos convertido la caja en un contenedor, nos podemos ayudar con el inspector de elementos de chrome, para identificar las lineas y los items de esta grilla.
Pero aparte de esto la estructura sigue siendo igual. Una caja debajo de otra.
Cómo lo cambiamos? 


                                .contenedor {
                                    ...
                                    grid-template-columns: 100px 200px 300px;
                                }

Con esta propiedad le damos un ancho especifico a cada columna, en este caso solo tenemos 3 tamaños, eso quiere decir que solo vamos a tener 3 columnas. Pero en este caso tenemos 6 elementos, que sucede?
Al terminar de colocar items en 3 columnas, entonces se crea otra fila para los elementos restantes
Pero como le damos tamaños a las filas? 


                                .contenedor {
                                    ...
                                    grid-template-rows: 150px 300px;
                                }

En este caso estamos dando 2 valores porque sabemos que solo se van a necesitar 2 filas, pero como damos un valor para todas las filas nuevas que se vayan creando. 


                                .contenedor {
                                    ...
                                    /* grid-template-rows: 150px 300px; */
                                    grid-auto-rows: 150px;
                                }

En este caso, vemos que cada item se va posicionando segun las columnas, si en algun caso no quisieramos este comportamiento, tendriamos que tener en cuenta entonces otras propiedades. 


                                .contenedor {
                                    border: 5px solid #e1bee7;
                                    background-color: #fff1ff;
                                    display: grid;
                                    /* grid-template-columns: 100px 200px 300px; */
                                    /* grid-template-rows: 150px 300px; */
                                    /* grid-auto-rows: 150px; */
                                    grid-auto-flow: column;
                                    grid-template-rows: 100px 200px 300px;
                                    grid-auto-columns: 150px;
                                }

En si la que hace este efecto es la propiedad grid-auto-flow, cuyo valor por defecto es row. Al especificarle el valor column, es que logramos esa estructura.

Aprendamos ahora a generar un espaciado entre items. 


                                .contenedor {
                                    ...
                                    row-gap: 10px;
                                    column-gap: 20px;
                                }

Y si queremos que el espaciado sea igual entre filas y columnas, lo podemos hacer en una sola propiedad. 


                                .contenedor {
                                    ...
                                    /* row-gap: 10px; */
                                    /* column-gap: 20px; */
                                    gap: 15px;
                                }

Sección 4 de 13 - Propiedades de la alineación

Propiedades de alineacion de los items.
Son las propiedades para alinear nuestros items en nuestra grilla.

  • justify-items
  • align-items
  • place-items

Recordar: justify es horizontal, align es vertical y place es una mezcla.

Propiedades de alineacion del contenedor.
Son las propiedades que ajustan la grilla en el espacio en el que vive

  • justify-content
  • align-content
  • place-content

Propiedades de alineacion del item - individual.

  • justify-self
  • align-self
  • place-self

                                .contenedor {
                                    border: 5px solid #e1bee7;
                                    background-color: #fff1ff;
                                    height: 600px;
                                    display: grid;
                                    gap: 15px;
                                    grid-template-columns: 150px 150px 150px;
                                    grid-auto-rows: 150px;
                                }
                                
                                .item {
                                    border: 5px solid #00bcd4;
                                    font-size: 4rem;
                                }

Tenemos un contenedor con los conceptos ya aprendidos, pero vamos a jugar con la alineacion. 


                                .contenedor {
                                    ...

                                    /* propiedades para alineacion contenedor */

                                    /* justify-content: center; */
                                    /* align-content: end; */
                                    place-content: center;
                                }

Aqui jugamos con la alineacion de toda la grilla 

                        
                            .contenedor {
                                ...
                            
                                /* Propiedades para la alineacion de los items (TODOS) */

                                /* justify-items: start; */
                                /* align-items: center; */
                                place-items: center;
                            }
